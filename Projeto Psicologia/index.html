Executando código do Python

Python é uma linguagem interpretada que reduz o ciclo de edição-teste-depuração porque não há necessidade de uma etapa de compilação. Para executar aplicativos de Python, você precisa de um ambiente de runtime/interpretador para executar o código.

A maioria dos ambientes de runtime dá suporte a duas maneiras de executar código Python:

1. Modo Interativo: nesse modo, cada comando que você digita é interpretado e executado imediatamente e você vê os resultados sempre que pressiona ENTER. É o modo padrão se você não passar um nome de arquivo para o interpretador.

2. Modo do script: nesse modo, você coloca um conjunto de instruções do Python em um arquivo de texto com uma extensão .py . Em seguida, você executa o interpretador python e o aponta para o arquivo. O programa é executado linha a linha e a saída é exibida. Não há nenhuma etapa de compilação, conforme mostra o diagrama a seguir.

Introdução: Entrada e Saída de dados.

ENTRADA DE DADOS

Entrada de Dados refere-se a entrada de dados do mundo externo para o meio digital. É dessa forma que enviamos informações para dentro das nossas aplicações. A forma mais primitiva de enviar dados para uma aplicação é pelo Prompt de Comando, isto é, pelo Console propriamente dito.

Uma aplicação pode, a qualquer momento, pedir que o usuário informe uma determinada informação, para isto, basta invocar a função input().

A partir da versão 3 do Python, a função input(), tem por objetivo de escrever a String passada como parâmetro e em seguida, ativar o modo de digitação, isto é, colocar o Console de uma determinada forma em que seja possível a digitação (entrada de dados). É importante observar que versões anteriores do Python funcionavam de maneira distinta, e também, que a função input() nas versões 2x era utilizado para outro finalidade!

A seguir, temos um exemplo onde imprimimos uma frase na tela e em seguida, é aberto para o usuário, a capacidade para digitação de letras e números.

input("Escreva entre aspas alguma informação a ser impressa no Console ")

O código acima, quando executado, irá imprimir o que está entre parêntesis no Console e em seguida, permitira ao usuário a digitação de qualquer tipo de dado alfanumérico. O conjunto de caracteres digitado pelo usuário, será retornado pela função input(). É responsabilidade do programador armazenar esses caracteres numa variável, como fazemos no exemplo a seguir:

#coding: utf-8

num = input("Digite um número:")
print(num)

No exemplo acima, será impresso a frase definida entre aspas e então, o console entrará num modo que permitirá o usuário digitar texto e números. Após o usuário digitar o que desejar e pressionar a tecla Enter, os dados digitados serão retornados pela função input(). No exemplo acima, estamos atribuindo o retorno da função para a variável num, logo, a mesma receberá aquilo que o usuário digitou no Console antes de pressionar a tecla Enter.

Por fim, imprimimos na tela os caracteres digitados pelo nosso usuário através da função print().

Nesta terceira parte iremos dar inicio a aula de estruturas condicionais.
Constantemente, acabamos lidando com situações onde precisamos tomar decisões, inclusive quando estamos a criar um programa para resolver algum problema. Escrever um programa onde seja necessário escolher entre uma situação ou outra(s) é muito comum no cotidiano de um programador.

ESTRUTURA DE DECISÃO IF..ELSE

É a estrutura de controle mais simples, que permite criar uma rotina onde execute um entre diversos agrupamentos de instruções. O bloco a ser executado é decidido por uma condição que pode ser uma expressão logica ou numérica, utilizando os operadores condicionais vistos na aula anterior, para fazer as comparações dos valores repassados que retornam um resultado que pode ser verdadeiro ou falso.
O IF possui duas formas em que pode se apresentar: Condicional simples e a condicional composta
Condicional Simples: A estrutura de decisão permite que um bloco de instruções seja executado (ou não) de acordo com o valor da condição que pode ser verdadeiro ou falso.
Em Python, a sintaxe do IF simples é representada assim:

#sintaxe do IF simples
if expressao:
   comando

Condicional Composta: Essa estrutura executa um comando quando a condição for verdadeira e outra condição quando for falsa. Sendo assim, faz a escolha de um entre dois blocos de instruções.

#sintaxe do IF composto
if expressao:
   comando
else:
   comando

Condicionais alinhadas

Comumente, durante o processo de desenvolvimento, encontramos problemas que não podem ser solucionados com condicionais simples, como por exemplo, se quisermos checar se um número está entre um valor e outro.
Como dito anteriormente, condicionais possuem a seguinte estrutura:
if expressão:
   comando

else:
   comando
Como fazer uso dessa estrutura para checar condições mais complexas? Através de condicionais aninhadas. Basicamente, condicionais aninhadas são condicionais dentro de outras condicionais, ou seja, condições que só são verificadas caso outra condição seja verdadeira.
Exemplos:

1. Escreva um programa que checa se um número está entre 0 e 10:

numero = int(input())
if numero < 10:
   if numero > 10:
      print("O numero está entre 0 e 10")
2. Escreva um programa que checa se um número é par, e se for, se é maior que 100:
numero = int(input())
if numero % 2 == 0:
   if numero > 100:
      print("O número é maior que 100")
 3. Escreva um programa que checa se um número é par ou ímpar, se for par, checa se ele está entre 100 e 200, se for ímpar, checa se o número é menor que 0:
numero = int(input())

if numero % 2 == 1:
   if numero < 0:
      print("O número é menor que 0");

else:
   if numero > 100:
      if numero < 200:
         print("O número está entre 100 e 200")
Um problema das condicionais aninhadas é que elas tendem a ser um pouco difíceis de ler, por isso, é ideal que se faça o uso, em alguns casos, da estrutura condicional composta, que possui a seguinte estrutura:

if condicao1:
   comando

elif condicao2:
   comando

else:
   comando
 A instrução “ELIF” executa somente se as condições acima foram falsas, elas age como se fossem uma instrução if aninhada dentro de uma instrução else, ou seja, é uma contração de “else if”

Outra forma de simplificar o código é através do operador lógico AND, que compara dois valores e retorna um valor verdadeiro se os dois forem verdadeiros, ela pode ser utilizada da seguinte forma:

if condicao1 and condicao2:
   comando
Agora, reescrevendo o exemplo 3 utilizando essas simplificações, se obtém o seguinte trecho de codigo:

if numero % 2 == 1 and numero < 0:
   print("O número é menor que 0")
elif numero < 200 and numero > 100:
   print("O número está entre 100 e 200")

Estrutura de Repetição

Existem situações em que é comum a necessidade de uma mesma instrução (ou conjunto delas) ser executada diversas vezes repetidamente. Nesta seção será apresentada a estrutura “ for ”.

ESTRUTURAS DE REPETIÇÃO: “ FOR ”

A estrutura “for” é caracterizada por obrigar a definição explícita em seu cabeçalho da quantidade de ciclos de execução sendo esses ciclos determinados pela quantidade de elementos contido na lista (strings, arrays, tuplas) declarada junto com a instrução “for”.
A sintaxe do Python não possui a estrutura de repetição “for” tradicional, onde define-se uma variável, condição e incremento no cabeçalho da estrutura. A ideia, é o trabalho e manipulação da estrutura ser simples e compacta para percorrer todos elementos das coleções ou estruturas que contenham listas de objetos.
Em Python, a sintaxe do “for” é representada assim:
#sintaxe do FOR:
for variável in lista:
   comando
Também podemos usar o laço for com outras funções como por exemplo o range():

Laço de repetição While

Estrutura de repetição: while
A estrutura while repete um bloco de instrução enquanto a condição determinada no cabeçalho permanecer verdadeira.
Sintaxe em Python do comando while:
while condição:
comando1
comando2
comando3
…
Quando o laço while é executado sua condição é testada. Caso esta condição seja verdadeira seus blocos de instruções serão executados, quando o último comando é executado a condição do laço é testada novamente e seu bloco é novamente executado. O laço termina quando sua condição for falsa.

Exemplo:
contador = 0                  #variável de controle

while contador < 5:           #condição, enquanto o ‘contador’ for menor que 5
     print(contador)              #comando a ser executado, imprimir variável na tela
     contador = contador + 1      #incrementando variável de controle
É possível colocar o comando else para ser executado ao final do laço while:
contador = 0  

                 while contador < 5:
             print(contador)    
               contador = contador + 1  
  
   else:
   print("Laço while encerrado.")
Usando o comando break:
while True:
   print ("Olá mundo")
   break
 Neste exemplo não há condição de parada, mas sim a expressão booleana True, dessa forma, o laço se tornaria infinito. O comando break logo abaixo do comando interrompe a execução do bloco de código e do laço.
